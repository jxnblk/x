.x

Hyperminimal CSS flexbox grid & micro layout alternative

Github npm

A fundamentally different approach to handling layout in CSS

Instead of adding up columns, .x is based around content sizes and dividing up available space, which makes it perfect for responsive web design. No clearfix, white space, or display table hacks â€“ just a simple abstraction of the flexible layout module.

502 B
30 Rules
32 Selectors
116 Declarations

Features

  • Vertical centering
  • Baseline alignment
  • Bottom alignment
  • Flexbox-based resizing
  • Fraction-based widths
  • Wrapping for template-based iterations
  • Equal height stretching
  • Tiled layout for template-based iterations
  • Width-based breakpoints
