
## Basic

Apply `.x` to a parent container to align items on the x-axis.

```html
<div class="x">
  <div class="p2">auto</div>
  <div class="p2">auto</div>
</div>
```

## xg

Add `.xg` to make a child element grow to fill the space.

```html
<div class="x">
  <div class="xg p2">
    <h2>.xg</h2>
    <p>flex grow</p>
  </div>
  <div class="p2">
    <p>auto</p>
  </div>
</div>
```
